========================================
  CARA TEST OTP DI MOBILE (ACODE)
========================================

LANGKAH 1: START SERVER
------------------------
1. Buka Acode di HP
2. Klik file: BUKA_LOGIN.bat
3. Pilih "Run" atau "Execute"
4. Server akan jalan di: http://localhost:8000

LANGKAH 2: BUKA DI BROWSER HP
-----------------------------
1. Buka browser di HP (Chrome/Firefox)
2. Ketik: localhost:8000/login.html
3. Atau klik link yang muncul di terminal

LANGKAH 3: TEST REGISTRASI & OTP
---------------------------------
1. Klik "Daftar" di halaman login
2. Isi form:
   - Username: (terserah)
   - Email: test@gmail.com (atau email apa saja)
   - Password: (terserah)

3. Klik tombol "Daftar"

4. KODE OTP AKAN MUNCUL DI:
   ‚úÖ KOTAK HIJAU BESAR dengan kode 6 digit
   ‚úÖ Ukuran font besar (mudah dibaca di HP)
   ‚úÖ Animasi pulse (berkedip)
   ‚úÖ Auto-scroll ke kode OTP
   ‚úÖ Kode otomatis terisi di form

5. Klik "Verifikasi & Daftar"

6. Selesai! Akun terdaftar.

-----------------------
