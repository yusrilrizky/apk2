  LANGKAH-LANGKAH LENGKAP
========================================

STEP 1: Pastikan Server PHP Jalan
----------------------------------

1. Buka Command Prompt/Terminal
2. Masuk ke folder project web Anda
3. Jalankan server:

   php -S 0.0.0.0:8000

   PENTING: Gunakan 0.0.0.0 bukan 127.0.0.1!
   
   0.0.0.0 = bisa diakses dari HP
   127.0.0.1 = hanya bisa diakses dari komputer

4. Server jalan, jangan ditutup!

---

STEP 2: Test Server dari Browser Komputer
------------------------------------------

1. Buka browser di komputer
2. Akses: http://192.168.1.2:8000/login.html
3. Pastikan halaman login muncul!

Jika tidak muncul:
- Cek server PHP jalan
- Cek file login.html ada
- Cek port 8000 tidak dipakai aplikasi lain

---

STEP 3: Pastikan HP & Komputer 1 WiFi
--------------------------------------

‚úÖ HP dan komputer HARUS terhubung ke WiFi yang SAMA!

Cara cek:
- Komputer: Settings ‚Üí Network ‚Üí WiFi
- HP: Settings ‚Üí WiFi

Contoh:
‚úÖ Komputer: WiFi "Rumah123"
‚úÖ HP: WiFi "Rumah123"

‚ùå Komputer: WiFi "Rumah123"
‚ùå HP: Data Seluler (4G/5G)

---

STEP 4: Test dari Browser HP
-----------------------------

1. Buka browser di HP (Chrome/Firefox)
2. Akses: http://192.168.1.2:8000/login.html
3. Pastikan halaman login muncul!

Jika tidak muncul:
- Cek HP & komputer 1 WiFi
- Cek firewall tidak block port 8000
- Cek server PHP masih jalan

========================================
  TROUBLESHOOTING
========================================

‚ùå "Tidak Dapat Terhubung"
--------------------------

Cek:
1. Server PHP jalan?
   ‚Üí php -S 0.0.0.0:8000

2. HP & komputer 1 WiFi?
   ‚Üí Cek WiFi settings

3. URL sudah benar?
   ‚Üí http://192.168.1.2:8000/login.html

4. Test di browser HP dulu?
   ‚Üí Buka URL di Chrome HP

5. Firewall block?
   ‚Üí Matikan firewall sementara

---

‚ùå "ERR_CONNECTION_REFUSED"
---------------------------

Artinya: Server tidak jalan atau port salah

Solusi:
1. Pastikan server jalan: php -S 0.0.0.0:8000
2. Cek port 8000 tidak dipakai aplikasi lain
3. Restart server

---

‚ùå "ERR_CONNECTION_TIMED_OUT"
-----------------------------

Artinya: HP tidak bisa akses komputer

Solusi:
1. Pastikan HP & komputer 1 WiFi
2. Matikan firewall Windows sementara
3. Test ping dari HP ke komputer

---

‚ùå Halaman Blank/Putih
----------------------

Solusi:
1. Cek URL sudah benar
2. Cek server jalan
3. Test di browser HP dulu
4. Rebuild APK

========================================
  CEK FIREWALL WINDOWS
========================================

Firewall bisa block koneksi dari HP!

Cara matikan sementara:
1. Windows Security
2. Firewall & network protection
3. Private network ‚Üí OFF (sementara)

Atau buat rule:
1. Windows Defender Firewall
2. Advanced settings
3. Inbound Rules ‚Üí New Rule
4. Port ‚Üí TCP ‚Üí 8000
5. Allow connection

========================================
  CEK KONEKSI HP KE KOMPUTER
========================================

Test ping dari HP:

1. Install app "Network Utilities" di HP
2. Ping: 192.168.1.2
3. Jika berhasil: Reply from 192.168.1.2
4. Jika gagal: Request timeout

Jika gagal ping:
- Firewall block
- Beda WiFi
- Router block
